## Breaking Changes

:::note Обновление требует действий

Это major-версия. Перед обновлением ознакомьтесь с изменениями -- они могут нарушить работу ваших интеграций.

:::

API: переход на camelCase. Что изменилось: все поля в ответах API переименованы из `snake_case` в `camelCase`.

До (v1.x):

```json
{
  "user_id": "usr_123",
  "created_at": "2024-01-01T10:00:00Z",
  "is_active": true
}
```

После (v2.0):

```json
{
  "userId": "usr_123",
  "createdAt": "2024-01-01T10:00:00Z",
  "isActive": true
}
```

Обновите код, который читает поля из ответов API. Подробная инструкция -- в [руководстве по миграции](#).

## Новые возможности

### Тёмная тема

Приложение поддерживает тёмную тему. Она включается автоматически по системным настройкам устройства.

Изменить вручную: [cmd:Настройки] -> [cmd:Внешний вид] -> [cmd:Тема].

### Интеграция с Google Calendar

Задачи с дедлайнами синхронизируются с Google Calendar. Как подключить:

1. Перейдите в [cmd:Настройки] -> [cmd:Интеграции]

2. Нажмите [cmd:Подключить Google Calendar]

3. Выберите календарь для синхронизации

4. Нажмите [cmd:Сохранить]

:::tip Двусторонняя синхронизация

Изменение дедлайна в TaskBoard обновит событие в Calendar. Изменение события в Calendar **не** обновит задачу в TaskBoard -- это ожидаемое поведение.

:::

### Массовые действия с задачами

Выберите несколько задач с помощью чекбоксов и примените действие сразу ко всем:

-  Изменить статус

-  Назначить исполнителя

-  Установить дедлайн

-  Удалить

## Удалено

:::danger Legacy API v0 отключён

Если вы до сих пор используете API v0, интеграция перестанет работать. Немедленно переходите на v1 -- инструкция в [руководстве по миграции](#).

:::

Также удалён виджет «Активность за неделю» -- он заменён новым дашбордом аналитики.

## Как обновиться

#### Веб-версия

Обновление происходит автоматически. Если видите старый интерфейс -- очистите кэш браузера: [kbd:Ctrl+Shift+R] (Windows/Linux) или [kbd:Cmd+Shift+R] (macOS).

#### Мобильное приложение

Обновите через App Store или Google Play. Поиск по названию: TaskBoard.

#### Self-hosted

Следуйте [инструкции по обновлению](#). Перед обновлением обязательно сделайте резервную копию базы данных.
